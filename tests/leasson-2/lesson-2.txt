git init : Tạo thư mục .git trong thư mục hiện. Biến nó thành 1 Git repository. 
    Khởi tạo các vùng : Working Directory , Staging Area và Local Repository
git add <file name> : Di chuyển thay đổi cụ thể từ Working Directory vào Staging Area
    chuẩn bị cho commit tiếp theo
git add . : Di chuyển tất cả thay đổi trong Working Directory vào Staging Area . Bao
    gồm file mới , file đã sửa đổi
git commit -m"message" : Tạo snapshot của tất cả các thay đổi trong Staging Area lưu 
    vào Local Repository với message mô tả .
